# technical_task

## Running project

Clone project to your computer

	git clone https://github.com/Unex322/technical_task.git

Install require packages

  
  ```
$ cd technical_task
$ npm install 
$ npm install --save ng2 file upload multer file saver
$ npm run build
$ mkdir uploads

```
---
  
Run your server on your localhost

	npm start
	
Go to

	http://localhost:3000/
